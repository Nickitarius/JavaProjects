1. Распознать своего/противника на войне (Положительное срабатывание - обнаружен союзник). 

Несимметричность и опасность ошибки очевидны - по своим стрелять нельзя, по врагу - нужно. Ошибка приводит к потерям.

2. Поиск "прирожденного преступника" по Ломброзо (Положительное срабатывание - преступник) (Пример из антиутопии).

Несимметричность - против "прирожденного преступника" предпринимаются превентивные меры (формальные или неформальные).

Опасность ошибки - риск подвергнуть дискриминации "не-преступника".
